# Bot Creator UI

Una interfaz de usuario moderna para crear configuraciones de bots personalizados.

## Tecnologías

- React
- TypeScript
- Tailwind CSS
- Lucide Icons
- Vite

## Instalación

```bash
# Clonar el repositorio
git clone https://github.com/tu-usuario/anahuac-ia.git

# Instalar dependencias
cd anahuac-ia
npm install

# Iniciar servidor de desarrollo
npm run dev
```

## Características

- Interfaz paso a paso
- Tema claro/oscuro
- Configuración de acciones del bot
- Exportación a JSON
